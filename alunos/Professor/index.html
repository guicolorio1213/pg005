Sobre o Projeto

Esta página foi criada para demonstrar como organizar conteúdo básico usando HTML5 e CSS3. A ideia é manter o foco na legibilidade e na leveza dos arquivos.

Principais Características

  • Design Responsivo (ajusta no celular).
  • Uso de HTML5 e CSS3 para estrutura e estilo.
  • Imagens otimizadas para web.
  • Sem dependências externas, apenas código limpo.

Tecnologias Utilizadas

O projeto foi desenvolvido utilizando as seguintes tecnologias:

  • HTML5
  • CSS3
  • Bootstrap 5 (para estilos e componentes responsivos)
